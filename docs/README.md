# Documentacao do Projeto

Este diretorio contem toda a documentacao do projeto Forum Academico UNIFEI.

## Arquivos Disponiveis

- **STATUS_REPORT.md** - Relatorio de status do projeto
- **REGISTRO_DE_BUGS.md** - Registro completo de bugs
- **BUGS_MANTIS_EXPORT.csv** - Exportacao para Mantis Bug Tracker
- **BUGS_BUGZILLA_EXPORT.xml** - Exportacao para Bugzilla

## Status Report

O Status Report contem:
- Revisao de marco
- Acompanhamento de acoes corretivas
- Pendencias do periodo
- Parecer sobre o projeto
- Metricas completas
