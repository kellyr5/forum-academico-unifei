## Rastreamento de Bugs

### Sistemas de Controle de Bugs

Os bugs deste projeto foram registrados e rastreados usando os seguintes sistemas:

#### Mantis Bug Tracker
**Arquivo:** `docs/BUGS_MANTIS_EXPORT.csv`
- Formato: CSV (compatível com Mantis)
- Total de bugs: 5
- Status: Todos resolvidos
- Pode ser importado em qualquer instalação do Mantis

#### Bugzilla
**Arquivo:** `docs/BUGS_BUGZILLA_EXPORT.xml`
- Formato: XML (formato padrão Bugzilla)
- Total de bugs: 5
- Status: Todos resolvidos (FIXED)
- Pode ser importado em qualquer instalação do Bugzilla

#### Markdown (Legível)
**Arquivo:** `docs/REGISTRO_DE_BUGS.md`
- Formato: Markdown
- Documentação completa de cada bug
- Inclui passos para reproduzir e soluções

### Como Importar os Bugs

**Para Mantis:**
1. Acesse seu Mantis
2. Vá em "Manage" → "Import Issues"
3. Selecione o arquivo `BUGS_MANTIS_EXPORT.csv`
4. Clique em "Import"

**Para Bugzilla:**
1. Use a ferramenta `importxml.pl` do Bugzilla
2. Execute: `./importxml.pl BUGS_BUGZILLA_EXPORT.xml`
3. Ou importe via interface web (se disponível)
